# 按钮

```规则
    添加基础的类名x-button
    以x-button-* 的形式命名
```

````html
 <div class="x-button x-button-large x-button-blue">大号按钮</div>
    <div class="x-button">默认按钮</div>
    <div class="x-button x-button-small x-button-darkseagreen">小号按钮</div>
    <div class="x-button"><i class="iconfont icon-baobei"></i>带符号</div>
````
